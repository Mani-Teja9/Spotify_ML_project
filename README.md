# 🎵 Spotify Track Popularity & Genre Prediction

## 📌 Project Overview
This project focuses on analyzing Spotify track data to understand the key factors influencing music popularity and genre classification. Using supervised machine learning techniques, the project explores how audio features contribute to predicting track popularity and identifying genres.

---

## 📊 Dataset
- **Name:** Spotify Tracks Dataset  
- **Source:** https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset  
- **Size:** ~114,000 rows, 20+ features  
- **Type:** Structured dataset with numerical and categorical features  

---

## 🎯 Objectives
- Analyze relationships between audio features and popularity  
- Identify dominant music genres  
- Build models to predict track popularity  
- Explore classification of genres using audio features  

---

## 🔍 Research Questions
1. Which audio features most influence track popularity?  
2. Can popularity be predicted using audio features?  
3. What are the most common genres in the dataset?  
4. Can machine learning classify genres accurately?  
5. How does tempo affect popularity?  
6. Does danceability impact track success?  
7. Which features contribute most to predictions?  

---

## 📊 Results
- Audio features like **danceability, energy, and tempo** show influence on popularity  
- Genre distribution highlights dominant music categories  
- Basic regression model provides initial predictive insights  
- Classification model demonstrates feasibility of genre prediction
